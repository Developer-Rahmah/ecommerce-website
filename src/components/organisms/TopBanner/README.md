# TopBanner

Top promotional banner displaying key features like free returns, fast delivery, and quality fabrics.

## Usage

\`\`\`tsx
import { TopBanner } from '@/src/components/organisms/TopBanner';

<TopBanner />
\`\`\`

## Props

- `className` (string, optional): Additional CSS classes

## Features

- Responsive layout
- Icons for visual appeal
- Hides "Finest Fabrics" on mobile
- Dark background with white text
